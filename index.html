Mermaid Diagram Renderer

Render a Mermaid diagram using markdown from the browsers queryParams.
This web app makes is possible to share diagrams easily!

Usage 👇 Ensure you URL encode your markdown before setting queryParms like so:

C#: 

                $"?diagram={HttpUtility.UrlEncode(mermaid)}&title=Foo"
JS: 
                `?diagram=${encodeURI(myMermaidMarkdown)}&title=Foo`
